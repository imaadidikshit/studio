# **App Name**: Nebula AI

## Core Features:

- Hero Section with Glowing Orb: Visually striking hero section featuring a glowing background orb animation to capture user attention.
- AI Data Simulation: An interactive playground section simulating AI typing and data processing to demonstrate functionality.
- Bento Grid Feature Display: Showcase features in a dynamic Bento Grid layout, providing an engaging and visually diverse presentation.
- Glow Toggle Switch Pricing: Implement a pricing section with a toggle switch (Monthly/Yearly) and a glowing 'Most Popular' card to highlight value.
- Partner Logo Marquee: Display partner logos in an infinite scrolling marquee with grayscale-to-color hover effect for social proof.
- Magnetic CTA Buttons: Attractive call-to-action buttons with a magnetic effect, enhancing user engagement.

## Style Guidelines:

- Primary color: Neon purple (#6441FF) to create a glowing, futuristic effect.
- Background color: Deep void black (#0a0a0a) for a luxury dark mode theme.
- Accent color: Cyber blue (#4EECFF) to complement the primary neon purple and add contrast.
- Headline font: 'Clash Display' (serif) for a high-end, editorial look.
- Body font: 'Plus Jakarta Sans' (sans-serif) for a modern, techy feel. Note: currently only Google Fonts are supported.
- Use Glassmorphism with frosted glass blur, Bento Grid layouts, and subtle noise textures.
- Hover effects on buttons and cards, with glowing borders on hover; subtle fade-in animations on scroll.
